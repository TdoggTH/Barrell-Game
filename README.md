# Barrell-Game
First person action shooter game set in an arena with throwing barrels as the main gameplay mechanic.

The game is meant to inspire emergent gameplay and creative thinking through the use of the environment. The main way this will be achieved is with Barrels which will be:
 1. **Powerful**, Barrels will be inherently optimal to use.
 2. **Multi-Purpose**, Barrels will have multiple uses. 
 3. **Random**, Barrels spawn state will be randomised with limited exceptions so no two time playing will be the same.
